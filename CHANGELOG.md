# 版本说明

## v0.1.1 - 文字区域识别与画笔编辑稳定性

发布日期：2026-05-13

### 新增

- 增加标题/文字区域检测，自动分割时会优先给出可点击的 `标题文字`、`文字区域` 候选。
- 增加 Logo 场景处理：Logo 里的字会尽量和图形标识合并为一个 `品牌标识` 区域，而不是把字母单独拆成文字块。
- LLM 语义分割提示词同步更新：保留有意义的文字候选，但品牌 Logo 作为整体处理。

### 改进

- 画笔区域编辑请求改为只提交原图和黑白 mask，不再提交大号半透明 overlay，降低大图请求失败概率。
- 编辑生成时在右侧结果画板显示等待时长，明确提示本地 Codex CLI 通常需要 3-6 分钟。
- 编辑失败时在右侧画板显示具体错误原因，方便排查 provider、接口或 Codex CLI 问题。
- 后端增加编辑请求日志，记录 provider、原图、mask 和尺寸。

### 验证

- `npm run check` 通过。
- 已用画笔 mask 复现 `/api/edit`，本地 Codex CLI 成功返回 1672x941 编辑结果。

## v0.1.0 - 本地语义化 AI 修图 Demo

发布日期：2026-05-13

### 初始功能

- 本地 Web 画布，支持生成图片、上传图片、左右并排对比原图和修改结果。
- 支持本地 Codex CLI 作为默认图片生成/编辑 provider。
- 支持 OpenAI Images API 和自定义图片 API 配置。
- 支持 FastSAM、SAM 和轻量 fallback 自动分割。
- 支持 LLM 语义整理分割候选。
- 支持点击分割区域、画笔圈选、擦除、清空选区和画布缩放。
- 支持 macOS launchd 后台服务和自动重启脚本。
