# Social Preview Design Philosophy: Clean Semantic Compare

Clean Semantic Compare treats the launch image as a product demonstration, not a poster full of claims. The layout is split into a restrained identity block and one large proof card, so viewers understand the promise and see the before/after result without decoding overlapping fragments.

The right side carries the evidence. Instead of cropping the examples into decorative slices, the complete before/after comparison is presented as a single large card with generous margins. This makes the selected mask, generated result, and short instruction readable in GitHub, WeChat, and mobile feed previews.

The color system remains precise: green signals user selection, blue signals AI transformation, and warm off-white keeps the product approachable. The composition should look meticulously crafted, with disciplined spacing, calm shadows, and no competing labels.

Text is deliberately minimal: the name, the promise, one short edit prompt, and the install command. Everything else is carried by the large visual comparison.
